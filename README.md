# SkyCast 🌤️ | Real-Time Weather Application

A clean, responsive web application that provides real-time weather updates, forecasts, and climate metrics for locations worldwide. This project leverages external weather APIs to deliver accurate, up-to-the-minute environmental data in an intuitive interface.

## 🚀 Key Features
* **Live Weather Tracking:** Instant temperature, humidity, wind speed, and atmospheric pressure updates for any searched city.
* **Dynamic Weather Layouts:** UI backgrounds and icons that adapt automatically based on the current weather condition (sunny, rainy, snowy, overcast).
* **Location Auto-Detection:** (Optional) Uses browser geolocation to pull local weather immediately upon loading.
* **Responsive Layout:** Clean, mobile-first design optimized for mobile screens, tablets, and desktops.

---

## 🛠️ Built With
* **Frontend:** React.js / JavaScript (ES6+), HTML5, CSS3
* **API Integration:** OpenWeatherMap API (or the specific weather API you used)
* **Version Control:** Git & GitHub
